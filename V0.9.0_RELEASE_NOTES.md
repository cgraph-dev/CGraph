## 🚀 Getting Started

### For Users

#### Enable Rich Media:
1. Update to v0.9.0
2. Rich media automatically enabled
3. Send image/video/audio links in messages
4. Tap embeds to view fullscreen

#### Subscribe to Premium:
1. Go to Settings → Premium
2. Choose your tier (Free/Premium/Premium+)
3. Select billing cycle (Monthly/Yearly)
4. Complete purchase
5. Enjoy premium features instantly!

#### Shop for Coins:
1. Go to Shop → Coin Shop
2. Purchase coin bundles or claim daily bonus
3. Browse Themes, Badges, Effects, and Boosts
4. Purchase items with your coins
5. Items apply automatically

### For Developers

#### Installation:
```bash
# Update to latest version
git pull origin main
git checkout v0.9.0

# Install dependencies
pnpm install

# Run development server
pnpm dev
```

#### Import Components:
```typescript
// Rich Media Embeds
import { RichMediaEmbed } from '@/components';

// Premium Screens
import { PremiumScreen, CoinShopScreen } from '@/screens/premium';
```

#### Integrate in Messages:
```typescript
<RichMediaEmbed 
  content={message.content}
  isOwnMessage={message.sender.id === user?.id}
  onLoad={() => scrollToBottom()}
/>
```

#### Add to Navigation:
```typescript
<Stack.Screen 
  name="Premium" 
  component={PremiumScreen}
  options={{ title: 'CGraph Premium' }}
/>
<Stack.Screen 
  name="CoinShop" 
  component={CoinShopScreen}
  options={{ title: 'Coin Shop' }}
/>
```

## ⚠️ Important Notes

### Payment Integration Required

The premium features and coin shop are **ready for production** but require payment provider integration:

**Recommended:**
- **Mobile:** RevenueCat (cross-platform IAP)
- **Web:** Stripe (credit/debit cards)
- **iOS:** Apple In-App Purchases
- **Android:** Google Play Billing

**Setup Required:**
1. Create accounts with payment providers
2. Configure API keys in environment
3. Test purchase flows in sandbox
4. Submit for app store review
5. Deploy to production

### No Breaking Changes

Version 0.9.0 is **fully backward-compatible**. All existing functionality continues to work without changes.

### Testing Recommendations

Before deploying to production:
- ✅ Test on real iOS devices (iPhone 12+)
- ✅ Test on real Android devices (Android 11+)
- ✅ Test subscription purchase flows
- ✅ Test coin purchases and spending
- ✅ Test rich media with various URLs
- ✅ Test on different screen sizes
- ✅ Test offline functionality
- ✅ Test payment provider integration
