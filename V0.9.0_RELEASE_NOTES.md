# CGraph v0.9.0 Release Notes

**Release Date:** 2025-01-23  
**Status:** In Progress

---

## 🎯 Overview

Version 0.9.0 focuses on **mobile feature parity** - connecting all mobile feature hooks to their backend implementations and adding mobile WebRTC support. This release ensures web and mobile apps work together seamlessly.

## ✅ Completed Changes

### Package Version Sync
All internal packages are now synchronized to v0.8.6:
- `@cgraph/shared-types` - 0.7.58 → 0.8.6
- `@cgraph/ui` - 0.7.58 → 0.8.6
- `@cgraph/utils` - 0.7.58 → 0.8.6
- `@cgraph/config` - 0.7.58 → 0.8.6
- `@cgraph/core` - 0.8.3 → 0.8.6
- `@cgraph/state` - 0.8.3 → 0.8.6 (also fixed zustand ^4.4.0 → ^5.0.0)

### Mobile Authentication Hooks
Connected mobile auth hooks to existing AuthContext:

- **`useAuth()`** - Now wraps `AuthContext.useAuth()` with haptic feedback
- **`useBiometricAuth()`** - Biometric authentication (Face ID, Touch ID, fingerprint)
- **`useTwoFactor()`** - Enable/disable/verify 2FA with TOTP
- **`useSessions()`** - View and revoke active sessions

**File:** [apps/mobile/src/features/auth/hooks/index.ts](apps/mobile/src/features/auth/hooks/index.ts)

### Mobile Premium Hooks
Connected premium/subscription hooks to backend API and PaymentService:

- **`usePremiumStatus()`** - Check subscription tier, feature access
- **`useSubscription()`** - Subscribe, cancel, restore purchases
- **`useCoins()`** - Check balance, spend coins, purchase coin bundles
- **`useShop()`** - Browse and purchase shop items
- **`usePremiumHaptics()`** - Haptic feedback for premium actions

**File:** [apps/mobile/src/features/premium/hooks/index.ts](apps/mobile/src/features/premium/hooks/index.ts)

### Mobile Group Hooks
Connected group management hooks to backend API:

- **`useGroups()`** - Fetch user's groups
- **`useGroup(groupId)`** - Single group details, update, leave
- **`useChannels(groupId)`** - List and create channels
- **`useChannelNotifications(channelId)`** - Mute/unmute channels
- **`useGroupMembers(groupId)`** - List, kick, ban members
- **`useGroupInvites(groupId)`** - Create, share, revoke invites
- **`useJoinGroup()`** - Join via invite code or public groups
- **`useCreateGroup()`** - Create new groups

**File:** [apps/mobile/src/features/groups/hooks/index.ts](apps/mobile/src/features/groups/hooks/index.ts)

### Mobile WebRTC Infrastructure
Added WebRTC support for mobile voice/video calls:

- **`WebRTCManager`** - Full lifecycle management for calls
- **`useCall(socket)`** - React hook for call control
- **`useIncomingCallListener(socket, callback)`** - Listen for incoming calls

Features:
- Phoenix Channel signaling (same as web)
- ICE/STUN server support
- Multi-party mesh topology (up to 10 participants)
- Mute/unmute audio
- Enable/disable video
- Haptic feedback on all call events

**Files:**
- [apps/mobile/src/lib/webrtc/webrtcService.ts](apps/mobile/src/lib/webrtc/webrtcService.ts)
- [apps/mobile/src/lib/webrtc/useCall.ts](apps/mobile/src/lib/webrtc/useCall.ts)
- [apps/mobile/src/lib/webrtc/index.ts](apps/mobile/src/lib/webrtc/index.ts)

**Note:** Full WebRTC functionality requires `react-native-webrtc` package. For Expo managed workflow, you'll need a development build or expo-dev-client.

## 🔧 Technical Details

### Mobile Hook Architecture
```
AuthContext (contexts/AuthContext.tsx)
    └── useAuth() - Base auth state/methods
        └── features/auth/hooks/useAuth() - Adds haptic feedback
        └── features/auth/hooks/useBiometricAuth() - Adds biometrics
        └── features/auth/hooks/useTwoFactor() - Adds 2FA
        └── features/auth/hooks/useSessions() - Adds session mgmt

PaymentService (lib/payment.ts)
    └── features/premium/hooks/usePremiumStatus() - Status + hasFeature
    └── features/premium/hooks/useSubscription() - IAP/Stripe flow
    └── features/premium/hooks/useCoins() - Coin balance/purchase
    └── features/premium/hooks/useShop() - Shop items/inventory

API (lib/api.ts)
    └── features/groups/hooks/useGroups() - Group list
    └── features/groups/hooks/useGroup() - Group details
    └── features/groups/hooks/useChannels() - Channel list
    └── features/groups/hooks/useGroupMembers() - Member list
    └── features/groups/hooks/useGroupInvites() - Invite management
```

### WebRTC Call Flow
```
1. startCall(targetUserId) → Join call:lobby channel
2. create_room push → Get room_id
3. Join call:{room_id} channel
4. Peer joins → Exchange offers/answers via channel
5. ICE candidates exchanged → Connection established
6. endCall() → Leave channel, close peer connections
```

## 📦 Upgrade Guide

### For Mobile Developers

1. Update dependencies:
```bash
pnpm install
```

2. For full WebRTC support (voice/video calls):
```bash
cd apps/mobile
npx expo install react-native-webrtc
# Create development build
npx expo prebuild
npx expo run:ios  # or run:android
```

3. Use the new hooks:
```tsx
import { useAuth } from '@/features/auth/hooks';
import { usePremiumStatus, useCoins } from '@/features/premium/hooks';
import { useGroups, useGroupInvites } from '@/features/groups/hooks';
import { useCall } from '@/lib/webrtc';

function MyComponent() {
  const { user, login, logout } = useAuth();
  const { isPremium, hasFeature } = usePremiumStatus();
  const { balance, spend } = useCoins();
  const { groups, refresh } = useGroups();
  
  // ...
}
```

### For Backend Developers

No backend changes required for this release. All mobile hooks connect to existing API endpoints.
