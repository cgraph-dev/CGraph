# Claude Code Project Instructions

## Core Rules

1. **NEVER delete or remove any existing functions or features** - All existing code must be preserved
2. **Only make complex fixes with tested code** - Ensure all changes are properly tested
3. **Ensure compatibility with the whole project** - All changes must work with the entire codebase

## Development Guidelines

- When fixing bugs, preserve all existing functionality
- When adding features, do not remove or replace existing ones
- Before making changes, understand how the code integrates with the rest of the project
- Test changes to ensure they don't break existing features
- Prefer extending over replacing
- Keep backward compatibility at all times
