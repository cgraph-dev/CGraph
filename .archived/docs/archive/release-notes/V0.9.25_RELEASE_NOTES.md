# CGraph v0.9.25 Release Notes

**Release Date:** February 15, 2026

## Highlights

- **Compliance pass complete**: All backend modules under 500 lines, all React components under 300
  lines — enforced via sub-module + defdelegate (backend) and component extraction (frontend)
  patterns.
- **8 backend modules split** using the facade + sub-module architecture pattern, creating 18 new
  focused sub-module files.
- **5 React components split** by extracting sub-components and hooks, reducing the largest
  component from 425 to 293 lines.
- **Soft delete audit**: 45 `Repo.delete` calls reviewed; `delete_channel` fixed to use soft delete;
  intentional hard-deletes documented.
- **56 @spec annotations** added across 6 backend sub-modules for improved type safety.
- **Documentation overhaul**: Fixed broken cross-references, synchronized version numbers, updated
  architecture diagrams and feature counts across 9 docs.
