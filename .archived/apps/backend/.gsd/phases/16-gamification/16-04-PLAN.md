## Context

GAME-04/05/06/08 all depend on the XP pipeline (16-01) and quest system (16-02) being active. This
plan activates the competitive and economic features.

per-board scoping from 16-01's scoped leaderboard extensions. Web leaderboard widget (813 LOC) and
full page exist but only show global view.

`BattlePassTier` schema, tier progression, admin controller. Web events components (2,263 LOC).
Missing: auto-lifecycle worker and XP→tier-progress integration.

**Currency (GAME-06):** `award_coins`/`spend_coins` fully implemented with race protection. 16-01
adds companion coin awards from XP. Need: coin earning visibility, daily coin cap, coin reward
animations.

**Marketplace (GAME-08):** Full `Marketplace` context (224 LOC), `MarketplaceItem` schema.
Controller (366 LOC) + admin (381 LOC). Web marketplace page (965 LOC) + price history chart (869
LOC). Missing: atomic purchase transaction (coin transfer + ownership) and listing creation UX.

  defmodule CGraph.Gamification.EventLifecycleWorker do
    use Oban.Worker, queue: :gamification, max_attempts: 3
    
    alias CGraph.Gamification.Events
    
    @impl Oban.Worker
    def perform(%Oban.Job{args: %{"action" => "check_lifecycle"}}) do
      now = DateTime.utc_now()
      
      # Activate events whose starts_at has passed
      Events.Crud.list_upcoming_events()
      |> Enum.filter(fn e -> DateTime.compare(e.starts_at, now) != :gt end)
      |> Enum.each(&Events.Crud.activate_event/1)
      
      # End events whose ends_at has passed
      Events.Crud.list_active_events()
      |> Enum.filter(fn e -> DateTime.compare(e.ends_at, now) != :gt end)
      |> Enum.each(fn event ->
        Events.Crud.end_event(event)
        distribute_event_rewards(event)
        broadcast_event_ended(event)
      end)
      
      :ok
    end
    
    defp distribute_event_rewards(event) do
      # Top 10 get bonus rewards based on rank
      # All participants get participation rewards
    end
    
    defp broadcast_event_ended(event) do
      Phoenix.PubSub.broadcast(CGraph.PubSub, "gamification:events", {:event_ended, event.id})
    end
  end
  ```
  Register in Oban config: `{"*/15 * * * *", EventLifecycleWorker, args: %{action: "check_lifecycle"}}` (every 15 min)

  def purchase_listing(buyer, listing_id) do
    Repo.transaction(fn ->
      listing = Repo.get!(MarketplaceItem, listing_id) |> Repo.preload(:seller)
      
      # Validations
      if listing.listing_status != "active", do: Repo.rollback(:listing_not_available)
      if listing.seller_id == buyer.id, do: Repo.rollback(:cannot_buy_own_listing)
      
      # Atomic coin transfer
      {:ok, _buyer} = Gamification.spend_coins(buyer, listing.price, "marketplace_purchase",
        reference_type: "marketplace_item", reference_id: listing.id)
      
      seller = Repo.get!(User, listing.seller_id)
      {:ok, _seller} = Gamification.award_coins(seller, listing.price, "marketplace_sale",
        reference_type: "marketplace_item", reference_id: listing.id)
      
      # Transfer ownership
      transfer_item_ownership(listing, buyer)
      
      # Update listing status
      listing
      |> Ecto.Changeset.change(%{listing_status: "sold", buyer_id: buyer.id, sold_at: DateTime.utc_now()})
      |> Repo.update!()
      
      # Record in price history for chart
      record_price_history(listing)
      
      listing
    end)
  end
  ```
