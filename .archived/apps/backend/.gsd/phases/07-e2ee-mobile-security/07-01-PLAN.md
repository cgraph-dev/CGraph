Enable PQXDH + Triple Ratchet on web and auto-bootstrap E2EE after login.

E2EE-08 (key storage already done), and E2EE-09 (auto-bootstrap) for the web platform.

Encrypt-on-send (already working) now uses Triple Ratchet. </objective>
