# Plan 14-05: Real-Time Broadcasting Gaps + Integration Tests

## Context

The existing Phoenix Channels cover ~85% of real-time needs:

- `forum_channel.ex` (220L) — Forum-level: new_thread, thread_pinned/locked/deleted, member events,
  stats
- `thread_channel.ex` (406L) — Thread-level: new_comment, votes, typing, presence

Three gaps remain:

1. **No board-level channel** — When viewing a specific board, no channel exists to push new threads
   created in that board. The `forum_channel` broadcasts at the forum level, but boards need their
   own updates for large forums with many boards.
2. **No poll vote broadcasting** — `polls.ex` records votes but doesn't broadcast updated results to
   thread viewers.
3. **No post edit broadcasting** — `thread_posts.ex` updates posts but doesn't notify connected
   viewers.
4. **No reddit-style post creation broadcast** — `forum_channel` broadcasts thread events but not
   reddit-style post creation events.

**What exists:**

- `user_socket.ex` — Routes `forum:*` and `thread:*` topics to their channels
- `forum_channel.ex` (220L) — Forum scope with member tracking + broadcasts
- `thread_channel.ex` (406L) — Thread scope with typing + votes + comments
- Web `useForumSocket` (130L) — Handles forum-level events
- Web `useThreadSocket` (267L) — Handles thread-level events
- Mobile `useForumChannel` and `useThreadChannel` — Real-time hooks

### Task 3: Create web useBoardSocket hook

**Files:** `apps/web/src/modules/forums/hooks/useBoardSocket.ts`,
`apps/web/src/modules/forums/hooks/index.ts`

**Actions:**

1. Create `useBoardSocket(boardId: string)`:
   ```typescript
   const useBoardSocket = (boardId: string) => {
     const [threads, setThreads] = useState<Thread[]>([]);
     const [viewers, setViewers] = useState<number>(0);

     useEffect(() => {
       const channel = socketManager.joinBoard(boardId);
       channel.on('new_thread', (thread) => {
         setThreads(prev => [thread, ...prev]);
       });
       channel.on('thread_updated', (thread) => {
         setThreads(prev => prev.map(t => t.id === thread.id ? thread : t));
       });
       channel.on('thread_pinned', ...);
       channel.on('thread_locked', ...);
       channel.on('thread_deleted', (payload) => {
         setThreads(prev => prev.filter(t => t.id !== payload.thread_id));
       });
       // Presence for viewer count
       return () => channel.leave();
     }, [boardId]);

     return { threads, viewers, isConnected };
   };
   ```
2. Add `joinBoard` method to `socketManager` if not present
3. Export from hooks index
