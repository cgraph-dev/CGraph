# v0.9.25: Architecture Compliance Pass Complete

We've completed a comprehensive compliance pass across the entire CGraph codebase, enforcing our
file size limits: **&lt;500 lines for backend modules** and **&lt;300 lines for React components**.

<!-- truncate -->

## What Changed

### Backend: Sub-module + Defdelegate Pattern

Eight backend modules that exceeded the 500-line limit were refactored using our **facade +
sub-module** pattern. The original module becomes a thin facade that delegates to focused
sub-modules:

- **groups.ex** went from 1,342 → 423 lines, with 5 sub-modules handling channels, members, roles,
  invites, and emojis separately
- **notifications.ex** went from 711 → 238 lines, with sub-modules for queries, delivery, and push
  tokens
- Four more modules (**audit**, **uploads**, **admin**, **tier_limits**) similarly split

The public API remains unchanged — all existing function calls continue to work through
`defdelegate`.

### Frontend: Component & Hook Extraction

Five React components exceeded the 300-line limit and were split:

- **MessageBubble** (425 → 293 lines) — media rendering and thread badges extracted
- **Matrix3DEnvironment** (394 → 151 lines) — Three.js effects extracted to individual components
- **VoiceMessageRecorder** (327 → 136 lines) — recording logic extracted to a custom hook

### Soft Delete Compliance

We audited all 45 `Repo.delete` calls across the backend. One bug was found and fixed:
`delete_channel` was doing a hard delete despite the Channel schema having a `deleted_at` field.
Intentional hard-deletes (roles, emojis, private messages) are now documented with rationale
comments.

### Documentation Sync

Nine documentation files were updated to fix broken cross-references, synchronize version numbers,
and add the compliance pass results to our tracking tables.

## By the Numbers

- **8 backend modules** refactored (18 new sub-module files created)
- **5 React components** split (8 new files extracted)
- **56 @spec annotations** added for type safety
- **45 Repo.delete calls** audited (1 bug fixed)
- **43 files changed**, +3,391 / −3,251 lines
