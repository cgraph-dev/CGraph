**`ScheduledMessage` schema + context:**
- `schedule_message(sender_id, conversation_id, content, scheduled_at, opts)`:
  - Validate scheduled_at is in the future (minimum 1 minute from now, max 7 days ahead).
  - Validate user is member of conversation.
  - Creates DB record + enqueues Oban job with `scheduled_at` as `scheduled_at` option.
  - Returns `{:ok, scheduled_message}`.
- `cancel_scheduled_message(message_id, user_id)` — only sender can cancel. Sets status to "cancelled".
- `list_scheduled_messages(user_id, conversation_id)` — user's pending scheduled messages.
- `update_scheduled_message(message_id, user_id, attrs)` — edit content or reschedule.

**`SendScheduledMessage` Oban worker:**
- On perform: fetch scheduled_message, verify status is "pending".
- Send as regular message via `Messaging.send_message/4` (or direct to conversation channel).
- Broadcast to conversation channel via PubSub.
- Update status to "sent", set `sent_at`.
- On failure: set status to "failed", retry up to 3 times.

**`Poll` schema + context:**
- `create_poll(creator_id, conversation_id, question, options, opts)`:
  - Validate 2-10 options, question max 500 chars, each option max 100 chars.
  - Validate user is member of conversation.
  - Options stored as JSONB array with generated UUIDs.
  - Broadcasts `poll_created` to conversation channel.
- `vote(poll_id, user_id, option_id)`:
  - Validate poll is not closed.
  - For single-choice: replace existing vote. For multiple-choice: add vote.
  - Broadcast `poll_vote_updated` with current tallies (if not anonymous, include voter names).
- `retract_vote(poll_id, user_id, option_id)` — remove a vote.
- `close_poll(poll_id, user_id)` — only creator can close. Broadcasts `poll_closed` with final tally.
- `get_poll_results(poll_id)` — returns tallies per option.

**`PollVote` schema:** Simple join table with poll_id, user_id, option_id.

**Add delegation functions in `messaging.ex`:**
- `defdelegate schedule_message(sender_id, conv_id, content, scheduled_at, opts), to: ScheduledMessage`
- `defdelegate create_poll(creator_id, conv_id, question, options, opts), to: Poll`
etc.
