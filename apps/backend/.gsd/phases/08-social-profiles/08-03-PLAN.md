  <files>apps/backend/lib/cgraph/accounts/user.ex, apps/backend/priv/repo/migrations/TIMESTAMP_add_status_expires_at_to_users.exs, apps/backend/lib/cgraph_web/channels/presence_channel.ex, apps/backend/lib/cgraph/presence/tracker.ex</files>
  <action>
1. Create an Ecto migration to add `status_expires_at` (`:utc_datetime`) to the `users` table:
   - `mix ecto.gen.migration add_status_expires_at_to_users` in apps/backend
   - Add column: `alter table(:users) do add :status_expires_at, :utc_datetime, null: true end`

2. Update `apps/backend/lib/cgraph/accounts/user.ex`:
   - Add `field :status_expires_at, :utc_datetime` to the schema (near existing `custom_status` and `status_message` fields)
   - Add `:status_expires_at` to the profile_changeset cast list (alongside `:custom_status`)
   - Add `:status_message` to profile_changeset cast if not already there

3. Update `apps/backend/lib/cgraph_web/channels/presence_channel.ex` `handle_in("set_status")`:
   - After the existing `Presence.update_status/4` call, persist to DB:
     ```elixir
     # Persist status_message + expiry to DB for reconnection survival
     status_message = params["status_message"]
     expires_in = params["expires_in"]  # seconds from now, or nil for no expiry
     expires_at = if expires_in, do: DateTime.add(DateTime.utc_now(), expires_in, :second), else: nil
     
     user
     |> Ecto.Changeset.change(%{
       status_message: status_message,
       custom_status: status,
       status_expires_at: expires_at
     })
     |> CGraph.Repo.update()
     ```
   - Include `status_message` and `status_expires_at` in the `friend_status_changed` broadcast payload (status_message is already included)

4. Update `handle_info(:after_join)` in PresenceChannel:
   - After tracking the user, restore persisted status from the user record:
     ```elixir
     # Restore persisted status on reconnect
     if user.status_message && user.status_message != "" do
       # Check if expired
       if is_nil(user.status_expires_at) || DateTime.compare(user.status_expires_at, DateTime.utc_now()) == :gt do
         Presence.update_status(socket, user.id, "lobby", user.custom_status || "online")
         broadcast_to_friends(user.id, friend_ids, "friend_status_changed", %{
           user_id: user.id,
           status: user.custom_status || "online",
           status_message: user.status_message,
           updated_at: DateTime.utc_now() |> DateTime.to_iso8601()
         })
       else
         # Status expired while offline — clear it
         user |> Ecto.Changeset.change(%{status_message: nil, custom_status: nil, status_expires_at: nil}) |> CGraph.Repo.update()
       end
     end
     ```

5. Run the migration: `cd apps/backend && mix ecto.migrate`
  </action>
