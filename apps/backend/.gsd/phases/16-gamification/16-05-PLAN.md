# Plan 16-05: Cosmetics Rendering, Animated Borders & Title Propagation

## Context

GAME-07 requires cosmetics to work. GAME-11 requires animated borders to be PERFORMANT. GAME-12 requires titles to display THROUGHOUT the app. The backend fully supports all three — schemas, ownership, equip/unequip APIs all exist. Web has avatar border store (146 LOC + 309 LOC actions + 164 LOC getters) and title-badge component (894 LOC). What's missing is the actual border animation CSS/rendering and the title display integration across message bubbles, forum posts, leaderboard entries, etc.

**Performance is critical for GAME-11.** Discord limits their animated profile effects to GPU-composited properties only (transform, opacity). Meta's gaming overlays use requestAnimationFrame for complex effects. Google's Material Design motion principles apply: animations should be purposeful and efficient.

**Key insight:** The `AvatarBorder` schema defines 13 animation types (`none, static, pulse, rotate, shimmer, wave, breathe, spin, rainbow, particles, glow, flow, spark`). Types `none` and `static` need no CSS animation — only 11 types need keyframe implementations. The Avatar component (`components/user/avatar.tsx`) needs to wrap with the animated border (note: border rendering already exists in `modules/social/components/avatar/`). The InlineTitle component needs to be inserted into every username display location.

<objective>
Implement performant animated avatar border rendering (13 schema types; 11 needing keyframes), propagate equipped titles throughout the app, and wire cosmetic preview/purchase UI.

Purpose: Cosmetics are the primary coin sink and visual reward — they must look great and perform well.
Output: AnimatedBorder CSS/components, InlineTitle component, Avatar integration, cross-app title display.
</objective>

  Create CSS keyframe animations for all 11 animated border types. CRITICAL: only use GPU-composited properties (transform, opacity, filter) — never animate width, height, margin, padding, top/left.
  
  ```css
  /* Performance: all animations GPU-composited only */
  .animated-border {
    contain: layout style;
    will-change: transform;
    position: relative;
    border-radius: 50%;
  }
  
  @media (prefers-reduced-motion: reduce) {
    .animated-border { animation: none !important; }
    .animated-border::before, .animated-border::after { animation: none !important; }
  }
  
  /* 1. Pulse — gentle scale pulse */
  @keyframes border-pulse {
    0%, 100% { transform: scale(1); opacity: 1; }
    50% { transform: scale(1.05); opacity: 0.8; }
  }
  .border-pulse { animation: border-pulse 2s ease-in-out infinite; }
  
  /* 2. Rotate — continuous rotation */
  /* 3. Shimmer — traveling highlight band */
  /* 4. Wave — undulating scale wave */
  /* 5. Breathe — slow opacity/scale oscillation */
  /* 6. Spin — fast continuous rotation */
  /* 7. Rainbow — rotating hue on conic gradient border */
  @keyframes border-rainbow {
    0% { filter: hue-rotate(0deg); }
    100% { filter: hue-rotate(360deg); }
  }
  .border-rainbow::before {
    content: '';
    position: absolute; inset: -3px; border-radius: 50%;
    background: conic-gradient(red, yellow, lime, aqua, blue, magenta, red);
    z-index: -1;
    animation: border-rainbow 4s linear infinite;
  }
  
  /* 8. Particles — floating particle effect */
  /* 9. Glow — outer glow intensity cycle */
  @keyframes border-glow {
    0%, 100% { filter: drop-shadow(0 0 4px var(--border-color)); }
    50% { filter: drop-shadow(0 0 12px var(--border-color)); }
  }
  .border-glow { animation: border-glow 3s ease-in-out infinite; }
  
  /* 10. Flow — smooth color flow gradient */
  /* 11. Spark — brief spark flashes */
  /* Note: `none` and `static` types need no keyframes */
  ```

  Types `none` and `static` need no CSS — handle them in the React component (no animation class applied).
  Use CSS custom properties for colors so borders can be customized.
  Use `contain: layout style` on every animated element.
  Total: ~250 lines of optimized CSS.

    animationType: AnimationType; // 'none' | 'static' | 'pulse' | 'rotate' | 'shimmer' | ... 13 types

    borderWidth?: number;        // px, default 3
    size: number;                // avatar size in px
    children: React.ReactNode;   // avatar image

    className?: string;
  }
  
  export function AnimatedBorder({ animationType, borderColor, size, children, ...props }: AnimatedBorderProps) {
    const prefersReducedMotion = useMediaQuery('(prefers-reduced-motion: reduce)');
    const borderClass = prefersReducedMotion ? '' : `border-${animationType}`;

      <div 
        className={cn('animated-border', borderClass, props.className)}
        style={{ 
          '--border-color': borderColor,
          width: size + 6, height: size + 6

      >
        {children}
      </div>
    );
  }

  interface InlineTitleProps {
    title: { name: string; color: string; rarity: TitleRarity } | null;
    size?: 'sm' | 'md' | 'lg';

      </span>
    );
