# CGraph v0.7.45 Release Notes

**Release Date:** January 11, 2026  
**Release Type:** Major Feature Release  

## 🎮 Complete Gamification System

This release implements the complete gamification backend that was documented but not yet implemented. The frontend components now have full backend API support.

---

## ✨ New Features

### 1. XP & Level Progression System
- **XP Tracking**: Users earn XP for various activities
- **Level Calculation**: Polynomial curve (`100 * level^1.5`) for balanced progression
- **XP Multipliers**: Premium users get 1.5x-2x XP boost
- **XP Transaction History**: Complete audit trail of all XP gains

**API Endpoints:**
- `GET /api/v1/gamification/stats` - Get user's complete gamification stats
- `GET /api/v1/gamification/level-info` - Get detailed level progression info
- `GET /api/v1/gamification/xp/history` - Get XP transaction history

### 2. Achievement System
- **12 Default Achievements** across 6 categories:
  - Social: First Message, Conversation Starter, Social Butterfly
  - Content: Voice Heard, Prolific Writer, Helpful Soul
  - Mastery: Week Warrior, Month Master, Year Legend
  - Exploration: Explorer
  - Secret: Night Owl (hidden)
  - Legendary: Early Adopter
- **Progress Tracking**: Track progress toward each achievement
- **Rewards**: XP and coin rewards on unlock
- **Title Unlocks**: Some achievements unlock special titles

**API Endpoints:**
- `GET /api/v1/gamification/achievements` - List all achievements with progress
- `GET /api/v1/gamification/achievements/:id` - Get specific achievement details

### 3. Quest System
- **Daily Quests**: Daily Messenger, Daily Reader, Community Supporter
- **Weekly Quests**: Weekly Creator, Weekly Helper, Social Week
- **Progress Tracking**: Real-time objective completion tracking
- **Repeatable Quests**: Daily/weekly quests reset automatically
- **Expiration System**: Quests expire if not completed in time

**API Endpoints:**
- `GET /api/v1/quests` - List all available quests
- `GET /api/v1/quests/active` - Get user's active quests with progress
- `GET /api/v1/quests/daily` - Get today's daily quests
- `GET /api/v1/quests/weekly` - Get this week's weekly quests
- `GET /api/v1/quests/:id` - Get specific quest details
- `POST /api/v1/quests/:id/accept` - Accept a quest
- `POST /api/v1/quests/:id/claim` - Claim quest rewards

### 4. Daily Login Streak System
- **Streak Tracking**: Consecutive login day tracking
- **Coin Rewards**: Base 10 coins + 5 per day (max 100)
- **Longest Streak**: Track personal best streak
- **Automatic Reset**: Streak resets if a day is missed

**API Endpoints:**
- `POST /api/v1/gamification/streak/claim` - Claim daily streak bonus

### 5. Multi-Category Leaderboards
- **XP Leaderboard**: Ranked by total XP
- **Level Leaderboard**: Ranked by level (XP as tiebreaker)
- **Streak Leaderboard**: Ranked by current streak
- **Karma Leaderboard**: Ranked by karma points
- **Messages Leaderboard**: Ranked by total messages sent
- **Posts Leaderboard**: Ranked by total posts created
- **User Rank**: Get user's personal rank in any category

**API Endpoints:**
- `GET /api/v1/gamification/leaderboard/:category` - Get leaderboard for category

### 6. Title System
- **9 Default Titles**: From Newcomer to Legend
- **Unlock Methods**: Level-based, achievement-based, purchasable
- **Equip System**: Users can equip one title at a time
- **Rarity Tiers**: Common, Uncommon, Rare, Epic, Legendary

**API Endpoints:**
- `GET /api/v1/titles` - List all titles with ownership status
- `GET /api/v1/titles/owned` - Get user's owned titles
- `POST /api/v1/titles/:id/equip` - Equip a title
- `POST /api/v1/titles/unequip` - Unequip current title
- `POST /api/v1/titles/:id/purchase` - Purchase a title with coins

### 7. Virtual Currency (Coins)
- **Earning Methods**: Daily login, quests, achievements, level-up
- **Transaction History**: Complete audit trail
- **Balance Tracking**: Real-time coin balance

**API Endpoints:**
- `GET /api/v1/coins` - Get current coin balance
- `GET /api/v1/coins/history` - Get coin transaction history
- `GET /api/v1/coins/packages` - List coin purchase packages
- `GET /api/v1/coins/earn` - Get ways to earn coins

### 8. Coin Shop
- **11 Default Items** across 5 categories:
  - Themes: Midnight, Ocean Breeze, Forest Grove, Sunset Glow, Neon Dreams
  - Badges: Verified, Star, Fire
  - Effects: Sparkle Effect
  - Boosts: 2x XP (24h), 2x XP (7 days)
- **Item Types**: Consumable, Permanent, Subscription
- **Premium Exclusives**: Some items require premium subscription
- **Limited Edition**: Support for limited quantity items

**API Endpoints:**
- `GET /api/v1/shop` - List all shop items
- `GET /api/v1/shop/categories` - List shop categories
- `GET /api/v1/shop/:id` - Get specific item details
- `POST /api/v1/shop/:id/purchase` - Purchase an item
- `GET /api/v1/shop/purchases` - Get purchase history

### 9. Premium Subscription System
- **Three Tiers**:
  - Free: Basic features
  - Premium ($4.99/mo): 1.5x XP, 10% coin bonus, custom themes, priority support
  - Premium+ ($9.99/mo): 2x XP, 20% coin bonus, exclusive effects, early access
- **Feature Gates**: Premium-only features properly gated
- **Expiration Handling**: Automatic tier management

**API Endpoints:**
- `GET /api/v1/premium/status` - Get subscription status
- `GET /api/v1/premium/tiers` - List all subscription tiers
- `GET /api/v1/premium/features` - Get user's available features
- `POST /api/v1/premium/subscribe` - Subscribe to a tier
- `POST /api/v1/premium/cancel` - Cancel subscription

## 🗄️ Database Changes

### New Tables
- `achievements` - Achievement definitions
- `user_achievements` - User achievement progress and unlocks
- `quests` - Quest definitions
- `user_quests` - User quest progress and completions
- `titles` - Title definitions
- `user_titles` - User title unlocks
- `shop_items` - Shop item catalog
- `user_purchases` - User purchase history
- `coin_transactions` - Coin balance audit trail
- `xp_transactions` - XP gain audit trail

### Extended Users Table
Added gamification fields:
- `xp`, `level`, `coins`
- `streak_days`, `streak_last_claimed`, `streak_longest`
- `equipped_title_id`, `equipped_badge_ids`
- `subscription_tier`, `subscription_expires_at`
- `daily_bonus_claimed_at`
- `total_messages_sent`, `total_posts_created`

## 🚀 Upgrade Instructions

1. **Pull the latest changes**
   ```bash
   git pull origin main
   ```

2. **Run the migration**
   ```bash
   cd apps/backend
   mix ecto.migrate
   ```

3. **Restart the backend**
   ```bash
   mix phx.server
   ```

4. **Test the endpoints**
   ```bash
   # Get gamification stats
   curl -H "Authorization: Bearer $TOKEN" http://localhost:4000/api/v1/gamification/stats
   
   # Claim daily streak
   curl -X POST -H "Authorization: Bearer $TOKEN" http://localhost:4000/api/v1/gamification/streak/claim
   
   # Get achievements
   curl -H "Authorization: Bearer $TOKEN" http://localhost:4000/api/v1/gamification/achievements
   ```
