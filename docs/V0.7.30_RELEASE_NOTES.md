# CGraph v0.7.30 Release Notes

**Release Date:** January 9, 2026

This release completes the developer experience improvements started in v0.7.29 by adding Storybook for interactive component documentation. All core UI components now have comprehensive stories with interactive controls, making it easier for developers to understand and use the component library.

---

## ✨ New Features

### Storybook Integration

Storybook provides an isolated environment for developing and documenting UI components. It's the industry standard for component-driven development and makes onboarding new developers much faster.

**Setup:**
```bash
cd apps/web
pnpm storybook      # Start dev server on port 6006
pnpm build-storybook # Build static documentation
```

**Configuration:**

The Storybook setup uses the latest v10.x with Vite for fast HMR:

| Package | Version | Purpose |
|---------|---------|---------|
| `storybook` | 10.1.x | Core framework |
| `@storybook/react-vite` | 10.1.x | React/Vite integration |
| `@storybook/addon-essentials` | 8.6.x | Controls, actions, docs |
| `@storybook/addon-interactions` | 8.6.x | Interactive testing |
| `@storybook/addon-links` | 10.1.x | Story navigation |
| `@storybook/test` | 8.6.x | Testing utilities |

**Features:**
- **Autodocs** — Automatic documentation generation from prop types
- **Controls** — Interactive prop editing in the sidebar
- **Actions** — Event logging for callbacks
- **Dark theme** — Matches the CGraph application theme
- **TypeScript** — Full type safety with `satisfies Meta<typeof Component>`

## 📝 Writing Component Stories

Follow this pattern for new components:

```tsx
import type { Meta, StoryObj } from '@storybook/react';
import { fn } from '@storybook/test';
import { MyComponent } from './MyComponent';

const meta = {
  title: 'Components/MyComponent',
  component: MyComponent,
  parameters: {
    layout: 'centered',
    docs: {
      description: {
        component: 'Description of the component.',
      },
    },
  },
  tags: ['autodocs'],
  argTypes: {
    variant: {
      control: 'select',
      options: ['primary', 'secondary'],
    },
  },
  args: {
    onClick: fn(), // Mock functions with fn()
  },
} satisfies Meta<typeof MyComponent>;

export default meta;
type Story = StoryObj<typeof meta>;

export const Default: Story = {
  args: {
    children: 'Click me',
  },
};

export const Interactive: Story = {
  render: function InteractiveComponent() {
    const [state, setState] = useState(false);
    return <MyComponent value={state} onChange={setState} />;
  },
};
```

## 🔧 Configuration Files

### .storybook/main.ts

```typescript
import type { StorybookConfig } from '@storybook/react-vite';
import path from 'path';

const config: StorybookConfig = {
  stories: ['../src/**/*.stories.@(js|jsx|ts|tsx)'],
  addons: [
    '@storybook/addon-essentials',
    '@storybook/addon-interactions',
    '@storybook/addon-links',
  ],
  framework: {
    name: '@storybook/react-vite',
    options: {},
  },
  viteFinal: async (config) => ({
    ...config,
    resolve: {
      ...config.resolve,
      alias: {
        ...config.resolve?.alias,
        '@': path.resolve(__dirname, '../src'),
      },
    },
  }),
  typescript: {
    reactDocgen: 'react-docgen-typescript',
  },
};

export default config;
```

### .storybook/preview.ts

```typescript
import type { Preview } from '@storybook/react';
import '../src/index.css';

const preview: Preview = {
  parameters: {
    layout: 'centered',
    backgrounds: {
      default: 'dark',
      values: [
        { name: 'dark', value: '#1a1a2e' },
        { name: 'light', value: '#ffffff' },
      ],
    },
    controls: {
      matchers: {
        color: /(background|color)$/i,
        date: /Date$/i,
      },
    },
  },
  tags: ['autodocs'],
};

export default preview;
```
