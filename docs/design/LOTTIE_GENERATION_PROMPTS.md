# Lottie Border Generation Prompts

> **Note:** FREE and COMMON borders (IDs 1–12) are **static** and should be generated as
> still PNG/SVG art, not animations. They will use a static `<Image>` / `<View>` component
> (`StaticRing`) instead of `LottieView` to save performance. You can also create
> single-frame Lottie JSONs for consistency.
>
> For RARE through MYTHIC borders (IDs 13–42), paste each prompt below into
> [LottieFiles AI](https://lottiefiles.com/ai), Rive, or a similar tool.
> Download the resulting `.json` file and place it into:
> - `apps/mobile/src/assets/lottie/borders/` (filename must match exactly)
> - `apps/web/src/assets/lottie/borders/` (same file, copied)
