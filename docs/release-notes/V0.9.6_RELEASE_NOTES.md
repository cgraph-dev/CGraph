# CGraph v0.9.6 Release Notes

**Release Date:** January 27, 2026

## Highlights

- **Avatar borders everywhere**: Unified `ThemedAvatar` usage across leaderboards, forums, chat,
  marketplace, events, members, and messaging surfaces for consistent border rendering.
- **Customization data parity**: Added avatar border fields to forum, marketplace, and hosting data
  models for consistent API → UI mapping.
- **Deployment readiness**: Verified Vite production build for web app and updated deployment
  documentation.

## Frontend Updates

- Standardized avatar rendering with border IDs in high-visibility UI surfaces.
- Fixed `ReferralDashboard` build issues and restored computed state with memoized helpers.

## Backend Updates

- Avatar border and chat customization schema migrations are included and ready to run.

## Build & Deployment

- **Web build:** `pnpm turbo run build --filter=@cgraph/web` ✅
- **Notes:** Vite chunk size warnings may appear in the build output; these are non-blocking.
