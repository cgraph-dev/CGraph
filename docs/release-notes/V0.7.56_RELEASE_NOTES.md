## рџЋЇ API Endpoints Expected

### Calendar
- `GET /api/v1/calendar/events` - Fetch events
- `GET /api/v1/calendar/events/:id` - Get single event
- `POST /api/v1/calendar/events` - Create event
- `PUT /api/v1/calendar/events/:id` - Update event
- `DELETE /api/v1/calendar/events/:id` - Delete event
- `GET /api/v1/calendar/categories` - Get categories
- `POST /api/v1/calendar/events/:id/rsvp` - RSVP to event

### Referrals
- `GET /api/v1/referrals/code` - Get user's referral code
- `POST /api/v1/referrals/code/regenerate` - Regenerate code
- `GET /api/v1/referrals` - Get user's referrals
- `GET /api/v1/referrals/stats` - Get referral statistics
- `GET /api/v1/referrals/leaderboard` - Get leaderboard
- `GET /api/v1/referrals/rewards` - Get reward tiers
- `POST /api/v1/referrals/rewards/:id/claim` - Claim reward
- `GET /api/v1/referrals/validate/:code` - Validate referral code
- `POST /api/v1/referrals/apply` - Apply referral code

### Members
- `GET /api/v1/members` - List members with filters
- `GET /api/v1/user-groups` - Get user groups
- `GET /api/v1/presence/online` - Get online users
