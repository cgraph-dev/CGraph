## 🚀 Deployment Instructions

### For Development Team

1. **Pull Latest Code**

```bash
git pull origin main
git checkout v0.9.0
pnpm install
```

2. **Start Development Servers**

```bash
# Backend
cd apps/backend && mix phx.server

# Web
pnpm web

# Mobile
pnpm mobile
```

3. **Integrate Payment Provider**

```bash
# Install dependencies
pnpm add @stripe/stripe-react-native
# OR
pnpm add react-native-purchases

# Configure in .env
STRIPE_PUBLISHABLE_KEY=pk_test_...
REVENUECAT_API_KEY=...
```

4. **Add to Navigation**

```typescript
// apps/mobile/src/navigation/AppNavigator.tsx
import { PremiumScreen, CoinShopScreen } from '@/screens/premium';

<Stack.Screen
  name="Premium"
  component={PremiumScreen}
/>
<Stack.Screen
  name="CoinShop"
  component={CoinShopScreen}
/>
```

5. **Test on Devices**

```bash
# iOS
pnpm mobile --ios

# Android
pnpm mobile --android
```
