# v0.7.31 Release Notes

**Release Date:** January 9, 2026

Cross-platform Storybook support and version stabilization.

---

## Highlights

### 🎨 Storybook for React Native

CGraph mobile now has full Storybook integration! Develop and test mobile components in isolation,
right on your device or simulator.

```bash
cd apps/mobile
pnpm storybook
```

### 📦 Storybook Version Alignment

All Storybook packages are now aligned to v8.6.15 — the latest stable release with full addon
support. This provides a consistent experience across web and mobile.

### 🐛 TypeScript Fixes

Fixed two TypeScript errors that were causing build warnings:

- Added React import to `preview.tsx` for JSX support
- Updated `ProfilerWrapper` to include `'nested-update'` phase (React 18+)

## Technical Details

### Storybook Version Strategy

Storybook v10 was released, but the addon ecosystem hasn't caught up yet. The v10 addons are still
in development, so we're using v8.6.15 which is the latest stable with full addon support.

When v10 addons are released, we'll upgrade both web and mobile to v10.

### Package Changes

**Web (apps/web):**

- `storybook` → 8.6.15
- `@storybook/react` → 8.6.15
- `@storybook/react-vite` → 8.6.15
- `@storybook/addon-essentials` → 8.6.15
- `@storybook/addon-interactions` → 8.6.15
- `@storybook/addon-links` → 8.6.15
- `@storybook/blocks` → 8.6.15
- `@storybook/test` → 8.6.15

**Mobile (apps/mobile):**

- `storybook` → 8.6.15 (NEW)
- `@storybook/react` → 8.6.15 (NEW)
- `@storybook/react-native` → 8.6.1 (NEW)
- `@storybook/addon-ondevice-controls` → 8.6.1 (NEW)
- `@storybook/addon-ondevice-actions` → 8.6.1 (NEW)
