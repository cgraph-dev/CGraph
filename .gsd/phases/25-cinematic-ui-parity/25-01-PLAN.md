<objective>
Extend @cgraph/animation-constants with shared presets for cinematic background effects and premium button animations.

Purpose: The landing page and auth pages use hardcoded animation values (particle counts, connection
distances, glow colors, magnetic pull strengths). To bring the same visual quality to web and mobile
apps without duplicating magic numbers, we centralize these as shared constants alongside the
existing springs/durations/easings.

Output:

- `backgrounds.ts` — presets for particle fields, cyber grids, aurora glows (counts, speeds, colors,
  connection distances)
- `buttons.ts` — presets for magnetic pull, shimmer, glow, flowing border animations
- Updated barrel export in `index.ts` </objective>
