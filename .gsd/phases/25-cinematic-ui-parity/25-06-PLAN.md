<objective>
Upgrade mobile app buttons to match the cinematic quality of landing/auth pages, adapted for native platform patterns.

Purpose: Mobile buttons currently use basic Reanimated spring press animation (scale 0.96→1). While
magnetic cursor-following doesn't apply to touch interfaces, buttons should still feel premium:
gradient animated borders, glow on press, enhanced haptic feedback patterns, and smooth spring
physics.

Output:

- `animated-button.tsx` — enhanced button with gradient border, glow press effect
- Updated `button.tsx` — composition of animated-button into existing variant system </objective>

  <files>apps/mobile/src/components/animated-button.tsx</files>
  <action>
    Create an enhanced button component using Reanimated:

    1. **Press animation**: Spring scale (0.96→1) using `springs.snappy` from animation-constants
    2. **Gradient border**: Use `react-native-linear-gradient` or `expo-linear-gradient` for animated gradient border (emerald→purple cycle)
    3. **Glow on press**: Animated shadow color/radius expansion on press (Reanimated `useAnimatedStyle`)
    4. **Haptic feedback**: Enhanced pattern — light impact on press, medium on release using `expo-haptics`

    7. Must work on both iOS and Android
    8. All animations on UI thread via `useAnimatedStyle` — zero JS thread work
