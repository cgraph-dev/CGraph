<objective>
Redesign web app avatars: change default shape from circle (rounded-full) to squircle (border-radius: 43px), and add Lottie animated avatar support.

Purpose: The user wants avatars to use a 43px border-radius "squircle" shape instead of the current
fully-circular (border-radius: 9999px) style. Additionally, avatars should support Lottie animations
for premium/gamification cosmetics — the infrastructure exists (lottie-web is installed, renderers
exist) but isn't wired into the avatar component.

Output:

- Updated avatar.tsx with squircle default and shape prop
- Lottie avatar rendering via existing lottie-renderer infrastructure
- Updated AnimatedAvatar border paths to work with squircle geometry </objective>
