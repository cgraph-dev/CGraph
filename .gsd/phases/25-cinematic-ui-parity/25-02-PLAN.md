Upgrade web app buttons to match landing page's cinematic button quality.

Purpose: Landing page buttons have magnetic pull (cursor-following displacement), idle shimmer
animation, and flowing gradient borders. Web app buttons currently only have basic spring tap/hover
scales. This plan closes that gap while keeping the existing button API backward compatible.

Output:

- `magnetic-button.tsx` — hook/wrapper for magnetic cursor-following behavior
- Enhanced `button.tsx` — primary/glass variants get magnetic + shimmer + flowing border
- CSS additions for shimmer keyframes </objective>

  <files>apps/web/src/index.css</files>
  <action>
    Add CSS keyframes and utility classes for button shimmer:

    ```css
    @keyframes btn-shimmer {
      0% { transform: translateX(-100%); }
      100% { transform: translateX(200%); }
    }
    .btn-shimmer::after {
      content: '';
      position: absolute;
      inset: 0;
      background: linear-gradient(90deg, transparent, rgba(255,255,255,0.08), transparent);
      animation: btn-shimmer 2s ease-in-out infinite;
      animation-delay: 4s;
    }
    @media (prefers-reduced-motion: reduce) {
      .btn-shimmer::after { animation: none; }
    }
    ```

    Add flowing border animation for primary buttons:
    ```css
    @keyframes btn-border-flow {
      0% { background-position: 0% 50%; }
      50% { background-position: 100% 50%; }
      100% { background-position: 0% 50%; }
    }
    ```
