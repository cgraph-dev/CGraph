  <action>
    Create modern forum browsing UI:

    **ThreadCard:**
    - Card layout with glass background (`bg-white/[0.04] border border-white/[0.06]`)
    - Content structure:
      - Top: Author avatar (24px) + username + timestamp + tag pills (colored)
      - Title: bold, 1-2 lines, linked to thread
      - Preview: first 2-3 lines of first post (muted text), truncated
      - Image preview: if thread contains images, show first image as thumbnail (right-aligned or bottom)
      - Bottom bar: vote count (with up/down arrows), reply count (chat icon), view count (eye icon), last active time
    - Hover state: elevated shadow, subtle border glow
    - Click navigates to thread
    - Pinned threads: pin icon + subtle brand border-left
    - Locked threads: lock icon, muted styling
    - Hot threads: fire emoji indicator based on recent activity

    **ThreadList:**
    - View toggle at top-right: Grid (2 columns of cards) | List (compact rows)
    - Sort options: Latest, Hot, Top (Today/Week/Month/All), Unanswered
    - Filter by tags (horizontal scrollable pills)
    - Grid view: CSS grid, 2 cols on desktop, 1 on tablet, gap using design tokens
    - List view: compact rows with title, author, votes, replies inline
    - Infinite scroll with skeleton loading
    - Empty state: "No threads yet — be the first to post"

    **VoteButton:**
    - Vertical layout: up arrow | count | down arrow (like Reddit)
    - Up arrow: brand color when upvoted, gray default
    - Down arrow: red when downvoted, gray default
    - Count animates on change (number rolls up/down using motion/react)
    - Click toggles vote, double-click (same direction) removes vote
    - Compact horizontal variant for use in list view

    **ForumSidebar (upgrade):**
    - Board categories as collapsible sections
    - Each board: icon + name + unread thread count badge
    - Active board highlighted
    - "Create Thread" button prominently at top
    - Board description shown on hover/click
    - Search boards

    **BoardHeader:**
    - Board name (large) + description + subscriber count + moderator list
    - Banner image if set
    - Action bar: Create Thread, Sort, View Mode, Search
    - Pinned threads row below header
    - Board rules collapsible card
  </action>

    All 5 files export named components.
  </verify>

  <action>
    Create post creation and viewing components plus mobile equivalents:

    **PostComposer (web):**
    - Rich text editor with toolbar: bold, italic, strikethrough, code, quote, bullet list, numbered list, heading, link, image, mention
    - Markdown preview toggle (side-by-side on desktop)
    - Image upload: drag-and-drop + click to upload, shows preview thumbnails
    - Tag selector: multi-select from board tags
    - Poll creation: toggle on to add poll options (min 2, max 10), single/multi vote, close date
    - Title field (required, max 120 chars)
    - Board selector if creating from non-board context
    - Post guidelines reminder card
    - Submit + Draft buttons
    - Character count indicator

    **PostView:**
    - Thread title (large, bold)
    - Author info: avatar + name + role badges + post date
    - Post content: rendered markdown with syntax highlighting, embedded images, quotes
    - Vote buttons at left side (vertical)
    - Action bar below: Reply, Quote, Share, Bookmark, Report
    - Reply chain: threaded replies with indent (2 levels max, then flat)
    - "Best answer" highlight if thread is Q&A type

    **PollCard:**
    - Renders within post content
    - Options as progress bars showing vote percentage
    - Own vote highlighted
    - Total voters count
    - Time remaining (if has expiry)
    - "Vote" button if not yet voted, changes to "Change vote" after voting
    - Results revealed after voting (or after close date)
    - Animated bar width transitions

    **Mobile ThreadCard:**
    - Single-column card matching web content but taller
    - Tap to navigate
    - Swipe left: bookmark
    - Vote buttons at left edge (compact)

    **Mobile ThreadList:**
    - FlatList of thread cards (single column only, no grid on mobile)
    - Pull-to-refresh
    - Sort/filter at top (horizontal pill row)
    - Infinite scroll

    **Mobile PostComposer:**
    - Full-screen modal when creating
    - Simplified toolbar (bold, italic, code, image, mention)
    - No side-by-side preview (preview toggle instead)
    - Image picker via expo-image-picker
    - Poll creation within
    - Submit from header (right button)

    **Mobile VoteButton:**
    - Horizontal layout (← count →)
    - Haptic feedback on vote
    - Animated count change
  </action>
