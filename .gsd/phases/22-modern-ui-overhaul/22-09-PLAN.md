  <action>
    Create premium navigation and notification components:

    **CommandPalette (new — Discord Ctrl+K style):**
    - Triggered by Ctrl+K / Cmd+K global keyboard shortcut
    - Modal overlay with large search input at top
    - Fuzzy search across: conversations, channels, users, forum threads, settings pages
    - Result categories: Recent, People, Channels, Threads, Settings
    - Each result: icon + primary text + secondary text (e.g., server name for channels)
    - Keyboard navigation: up/down arrows to select, Enter to navigate, Esc to close
    - Recent items shown before typing (last 5 navigated-to)
    - Actions available: "Go to #channel", "Message @user", "Open thread", "Search messages for..."
    - Animated entry: scale from center + backdrop blur
    - Fast: debounced search (150ms) with instant local results
    - Integrates with or replaces existing QuickSwitcher

    **SearchOverlay (upgrade):**
    - Full overlay or panel triggered from search icon (not same as command palette — this is for content search)
    - Search input with scope selector: "In this conversation", "In this server", "Everywhere"
    - Results grouped by type with section headers: Messages (show message + context), Users (avatar + name + status), Channels (icon + name + server), Forum Threads (title + votes)
    - Each result shows match highlights (bold matched text)
    - Filters: From user, In channel, Date range, Has attachment, Has link
    - Recent searches shown before typing
    - Infinite scroll results

    **SearchResultItem:**
    - Shared component for rendering a search result
    - Variants for each result type (message, user, channel, thread)
    - Message variant: shows message content with match highlights, author, timestamp, channel name
    - User variant: avatar + name + status + mutual friends
    - Thumbnail for attachment results

    **NotificationCenter (upgrade):**
    - Slide-out panel from right (or popover from bell icon)
    - Tabs at top: All, Mentions, Reactions, Follows
    - Notifications grouped by time: Today, Yesterday, This Week, Older
    - "Mark all as read" button in header
    - Each notification: icon (by type) + content + timestamp + avatar of actor
    - Unread notifications have subtle left brand-colored border
    - Click navigates to source (message, thread, profile)
    - Empty state per tab: "All caught up!" with illustration

    **NotificationItem:**
    - Shared item component for different notification types
    - Types: mention (@ icon), reaction (emoji), friend request (person+ icon), reply (arrow icon), like (heart icon), thread update (chat icon), system (bell icon)
    - Rich text: "**User** reacted ❤️ to your message in **#channel**"
    - Avatar of the actor on the left
    - Relative timestamp on the right
    - Swipe to dismiss (web: X button on hover)

    **NavigationSidebar (upgrade if needed):**
    - Top section: DMs button (with unread count), Friends button (with pending count)
    - Private channels section
    - User area at bottom (avatar, name, status, mic/headphone/settings)
    - Consistent with Discord's DM sidebar layout
    - Animated transitions between DM list and server sidebar
  </action>

  <action>
    Create call UI and mobile equivalents of search/notifications:

    **CallOverlay (web):**
    - Full-screen overlay when in a call (with PIP mode option)
    - Participant grid: adapts layout based on count (1: full screen, 2: side-by-side, 3-4: 2x2 grid, 5+: spotlight + sidebar)
    - Each participant: video feed (or avatar if camera off), name label at bottom, mute indicator icon
    - Speaking indicator: green border glow on active speaker
    - Screen share: shared screen gets spotlight (large), participants in sidebar strip
    - PIP mode: small draggable window in corner (320x180) when minimized
    - Background: dark/blurred, dismiss by clicking outside (with confirm if in call)

    **CallControls (web):**
    - Bottom bar in call overlay
    - Buttons: Mic toggle, Camera toggle, Screen Share, Reactions (emoji popup), Participants list, Chat, Settings gear, End Call (red)
    - Each button: icon + label below, toggle state (active = highlighted)
    - Reaction popup: floating emoji reactions that briefly appear on screen
    - Animated tooltips on hover
    - "You're muted" toast when trying to speak while muted (detected, if possible)

    **Mobile SearchOverlay:**
    - Full-screen search with back button
    - Search input at top with cancel
    - Results as scrollable list grouped by type
    - Recent searches section
    - Trending/suggested searches

    **Mobile NotificationCenter:**
    - Full screen (navigation destination, not overlay)
    - Same grouping as web: Today, Yesterday, This Week, Older
    - Swipe left on notification to dismiss
    - Pull-to-refresh
    - Unread count badge on tab bar icon

    **Mobile ExploreScreen (new — Instagram Explore-style):**
    - Grid layout of trending content: popular threads, active groups, featured users
    - Mixed content types in a masonry/staggered grid
    - Categories at top (horizontal pills): All, Groups, Forums, People, Events
    - Search bar at top (tap opens search overlay)
    - Tap on card navigates to content

    **Mobile CallControls:**
    - Bottom floating bar during call
    - Buttons: Mic, Camera, Speaker, Flip Camera, End Call (red, larger)
    - Swipe up to see participants
    - Incoming call: full-screen overlay with accept (green) / decline (red) buttons
    - Haptics on button press
  </action>
