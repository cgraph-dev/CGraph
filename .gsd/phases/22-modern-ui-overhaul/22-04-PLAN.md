  <action>
    Create mobile-native message interaction components:

    **SwipeActions (Reanimated + Gesture Handler):**
    - Swipe right to reply (Messenger-style): swipe reveals reply arrow icon, release triggers reply
    - Threshold: 60px to trigger, spring back animation
    - Haptic feedback (light impact) when threshold crossed
    - Optional: swipe left to delete (own messages only), with confirm
    - Uses `react-native-gesture-handler` Swipeable or custom pan gesture

    **MessageActionsSheet:**
    - Bottom sheet triggered by long-press on message
    - Haptic feedback (medium impact) on open
    - Quick reactions row at top: 👍 ❤️ 😂 😮 😢 🔥 (tappable emoji pills, horizontal scroll)
    - Action list below: Reply, Forward, Copy Text, Pin, Edit (own), Delete (own), Report
    - Each action has icon + label
    - Animated entry: slide up from bottom with backdrop dim
    - Uses Reanimated for sheet animation

    **ReactionBar:**
    - Horizontal row of reaction pills below message bubble
    - Each pill: emoji + count, highlighted if user reacted
    - Tap to toggle own reaction
    - Long-press to see who reacted (modal with user list)
    - "+" button to add new reaction (opens ReactionPickerSheet)
    - Pills animate in with scale spring

    **ReactionPickerSheet:**
    - Bottom sheet with emoji grid
    - Frequently used emojis at top
    - Category tabs (horizontal scroll)
    - Search bar
    - Tap emoji to add reaction, sheet dismisses

    **RichEmbed:**
    - Card below message bubble for detected links
    - Image thumbnail + site name + title + description
    - Tap opens URL in in-app browser (WebView or Linking.openURL)
    - Dismiss button

    **ThreadPreview:**
    - Compact pill: thread icon + "N Replies" + avatar stack + last reply time
    - Tap navigates to thread screen
    - Unread indicator in brand color
