  <action>
    Upgrade and create UI primitives to match Discord/Instagram quality:

    **Avatar (upgrade existing):**
    - Add `status` prop: `'online' | 'offline' | 'idle' | 'dnd' | 'invisible'` — renders a colored dot indicator on the bottom-right (green/gray/yellow/red/gray) exactly like Discord
    - Add `storyRing` prop: boolean — renders animated gradient ring around avatar like Instagram stories
    - Add `typing` prop: boolean — shows animated typing dots overlay
    - Sizes: `xs` (16px), `sm` (24px), `md` (32px), `lg` (40px), `xl` (56px), `2xl` (80px), `3xl` (120px)
    - Support `square` variant for server icons (rounded-lg instead of rounded-full) like Discord
    - Fallback: gradient background with initials (2 chars max), generate gradient from user ID hash

    **AvatarGroup (new):**
    - Stack up to N avatars with overlap (configurable via `max` prop, default 3)
    - Show `+N` overflow pill when exceeding max
    - Sizes inherit from Avatar `size` prop
    - Used for: group DM icons, participant lists, reaction user lists

    **StatusIndicator (new):**
    - Standalone status dot for use outside Avatar context
    - Supports `online|offline|idle|dnd|streaming` with appropriate colors
    - Optional pulse animation for `online` status

    **Skeleton (upgrade or create):**
    - Shapes: `text` (rectangle, 60-100% width randomized per line), `avatar` (circle), `card` (rounded rectangle), `message` (message-bubble shape with avatar), `thumbnail` (16:9 rectangle)
    - Animated shimmer effect (subtle left-to-right gradient sweep)
    - `count` prop for repeating — e.g., `<Skeleton shape="message" count={5} />`

    **ContextMenu (new, uses @radix-ui/react-context-menu):**
    - Animated entry with scale + fade (motion/react)
    - Sub-menu support
    - Keyboard icon hints (right-aligned, muted text)
    - Separator, label, and disabled states
    - Icons in menu items (left-aligned)
    - Used for: message right-click, channel right-click, user right-click

    **ScrollArea (new, uses @radix-ui/react-scroll-area):**
    - Custom thin scrollbar (4px, expands to 8px on hover) matching Discord's aesthetic
    - Auto-hide behavior with smooth fade
    - Both vertical and horizontal variants

    **Divider (new):**
    - Horizontal and vertical variants
    - Optional label in center (e.g., "January 5, 2026" date dividers in chat like Discord)
    - Subtle gradient fade at edges

    **Tooltip (upgrade):**
    - Add arrow pointer
    - Support `side` prop (top, right, bottom, left)
    - Compact dark background style like Discord tooltips
    - Delay on hover (300ms) to avoid flicker

    Export all new components from ui/index.ts barrel.
  </action>
