  <action>
    Create Discord-level message interaction components:

    **MessageActionsBar:**
    - Floating toolbar that appears top-right of message on hover (position: absolute)
    - Buttons: React (emoji face icon), Reply (arrow icon), Thread (chat bubble icon), More (three dots icon)
    - Compact pill shape with glass background (`bg-[var(--glass-bg)] border border-white/10 rounded-md shadow-lg`)
    - Each button has tooltip on hover
    - Animated entry: fade + translate-y using motion/react
    - "More" opens the context menu

    **MessageContextMenu (uses ContextMenu primitive from Plan 01):**
    - Items: Reply, Create Thread, Mark Unread, Copy Text, Copy Message Link, Pin Message, Edit (own messages only), Delete (own messages only), Report
    - Each item has left icon + label + optional keyboard hint
    - Separators between logical groups (actions | clipboard | moderation)
    - Conditional items: Edit/Delete only shown for own messages; Pin only for users with permission

    **ReactionPicker:**
    - Grid of frequently used emojis (top row) + searchable full emoji picker (Radix Popover)
    - Quick-react options: 👍 ❤️ 😂 😮 😢 🔥 (shown in action bar on hover)
    - Search input at top, category tabs (smileys, people, nature, food, activities, travel, objects, symbols)

    **ReactionBar:**
    - Renders below message content as horizontal row of pills
    - Each pill: emoji + count, highlighted if current user has reacted
    - Click to toggle own reaction, hover to see who reacted (tooltip with usernames, max 10)
    - "+" button at end to add new reaction (opens ReactionPicker)
    - Animated entry/counter change using motion/react

    **RichEmbed:**
    - Card rendered below message for detected links
    - Structure: colored left border (brand color or site-specific) | content area with: site name (small, muted), title (bold, linked), description (2 lines max), image thumbnail (right-aligned or bottom for large images)
    - Supports: Open Graph metadata, Twitter cards
    - Dismiss/collapse button (x icon top-right)
    - Max-width 520px like Discord embeds

    **ThreadPreview:**
    - Compact pill below message: thread icon + "N Replies" + avatar stack of last 3 repliers + last reply timestamp
    - Click opens thread panel
    - If thread has unread replies, show in brand color

    **TypingIndicator (upgrade):**
    - Shows at bottom of chat: "User is typing…", "User1 and User2 are typing…", "Several people are typing…"
    - Animated three dots (bouncing) using motion/react
    - Smooth enter/exit animation
  </action>
