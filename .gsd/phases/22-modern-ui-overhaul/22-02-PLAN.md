    - `fontWeight` object: normal ('400'), medium ('500'), semibold ('600'), bold ('700')
    - `letterSpacing` object: tight (-0.01 * fontSize), normal (0), wide (0.02 * fontSize)

    - 5 shadow levels (xs through xxl) using React Native shadow props (shadowColor, shadowOffset, shadowOpacity, shadowRadius) + Android elevation
    - Each level returns a StyleSheet-compatible object
