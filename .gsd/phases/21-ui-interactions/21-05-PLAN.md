Dialog currently uses conditional rendering: `if (!open) return null; return createPortal(...)`.
This prevents exit animations because the component unmounts instantly.

1. Add imports:

```ts
import { motion, AnimatePresence } from 'motion/react';
import { useReducedMotion } from '@/hooks/useReducedMotion';
import { transitions } from '@cgraph/animation-constants';
```

2. Replace the conditional rendering pattern with AnimatePresence:

Current pattern:

```ts
if (!open) return null;
return createPortal(<div className="overlay">...</div>, document.body);
```

New pattern:

```tsx
const reducedMotion = useReducedMotion();

return createPortal(
  <AnimatePresence>
    {open && (
      <>
        {/* Backdrop */}
        <motion.div
          key="dialog-backdrop"
          className="fixed inset-0 z-50 bg-black/50"
          initial={reducedMotion ? undefined : { opacity: 0 }}
          animate={{ opacity: 1 }}
          exit={{ opacity: 0 }}
          transition={{ duration: 0.2 }}
          onClick={() => onOpenChange(false)}
        />
        {/* Dialog content */}
        <motion.div
          key="dialog-content"
          className="fixed inset-0 z-50 flex items-center justify-center p-4"
          initial={reducedMotion ? undefined : transitions.modalSlideUp.initial}
          animate={transitions.modalSlideUp.animate}
          exit={transitions.modalSlideUp.exit}
          transition={reducedMotion ? { duration: 0 } : transitions.modalSlideUp.transition}
        >
          {children}
        </motion.div>
      </>
    )}
  </AnimatePresence>,
  document.body
);
```

3. Keep the useEffect for Escape key handling and body overflow lock — these are behavior, not
   animation.

- Do NOT change DialogProps interface
- Do NOT change DialogTitle, DialogContent, DialogActions sub-components (if they exist)
- Do NOT change the Escape key handler or body overflow logic
- Keep the createPortal target as document.body
- The backdrop uses `transitions.modalSlideUp` from @cgraph/animation-constants which has spring
  physics
