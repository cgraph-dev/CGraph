<objective>
Add spring-driven press and hover feedback to Button and IconButton components.

Purpose: The core Button and IconButton are plain HTML `<button>` elements with CSS-only
transitions. Adding `motion.button` with `whileTap` and `whileHover` provides immediate tactile
feedback that makes the entire app feel more responsive. These are the most-used interactive
elements (~1976 instances across the app).

Output: Both Button and IconButton wrapped with `motion.button`, using springs from useMotionSafe.
</objective>

Modify `apps/web/src/components/ui/button.tsx`:

import { motion } from 'motion/react';
import { useMotionSafe } from '@/hooks/useMotionSafe';

2. In the `Button` component function body, add:

```ts
const { tapScale, hoverScale, springs } = useMotionSafe();
```

3. Replace `<button` with `<motion.button` and add motion props:

```tsx
<motion.button
  ref={ref}
  disabled={disabled || isLoading}
  className={...}

  whileHover={hoverScale(1.02)}

  {...props}
>
```

   the press feedback. Keep all other CSS classes (hover:shadow-lg, hover:-translate-y-0.5, etc.) —
   only remove the CSS active:scale.
