<objective>
Enhance the onboarding wizard with "find friends" and "community" steps, and track completion server-side.

Purpose: AUTH-09 — onboarding wizard should guide new users through avatar → find friends → join community. The current web wizard has 4 steps (welcome/avatar → profile/bio → notifications → features) and the mobile has a simpler flow (avatar → name → bio → notifications). This plan adds the missing "find friends" and "community" steps, replaces/reorganizes existing steps to match AUTH-09, and adds server-side onboarding completion tracking so users don't re-enter the wizard.
Output: Updated onboarding flows on both platforms with friend discovery and community steps.
</objective>
